Tarea 7
Script de métricas para analizar desempeño de Aprendizaje no Supervisado
---------------------------------------------------------------------------------------

En la presente tarea se construyó el script de nombre MetricasNoSupervisado
el cual obtiene los valores de métricas Sum of Squares Within (SSW), 
Sum of Squares Between (SSB), WB-Index (WB) y Silhouette (SIL) para
algoritmos de aprendizaje no supervisado.

El script recibe como parámetros de entrada la matriz de datos y el vector
con el número de cluster asignado a cada observación a través de algoritmo 
no supervisado.

Para la prueba del script se trabajó con el dataset fisheriris con dos
descriptores: ancho de pétalo (en centímetros) y largo de pétalo (en centímetros).

Para visualizar los resultados abra el fichero de nombre DavidCevallos_Tarea7.mlx
y ejecute el live script.
