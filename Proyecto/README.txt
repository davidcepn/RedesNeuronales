-------------------------------------------------------------------------
Proyecto de Fin de Módulo Redes Neuronales
Chunk-based Malware Classifiers for Communication Networks
-------------------------------------------------------------------------

-------------------------------------------------------------------------
Realizado por: David Fabián Cevallos Salas
Fecha: 2023-08-15
-------------------------------------------------------------------------

El presente proyecto corresponde al análisis del dataset CIC-MalMem-2022
para la construcción de clasificadores de diferentes familias de malware
basadas en el método chunk hidding a través de varios algoritmos de
Machine Learning (ML).

Los entregables correspondientes al proyecto son:

- Paper 
- Código fuente Latex Overleaf de paper
- Notebooks con la solución:
	Notebook 1: Visualización tSNE de datos (Python)
	Notebook 2: Ejecución de algoritmos de ML (Python)
	Notebook 3: Implementación de redes neuronales (Python)
	Notebook 4: ANFIS (Matlab)
	Notebook 5: Aprendizaje No Supervisado (Matlab)
- Presentación Power Point

Acceda al repositorio GitHub del proyecto para obtener todos los ficheros
descritos anteriormente:

https://github.com/davidcepn/RedesNeuronales/tree/main/Proyecto
